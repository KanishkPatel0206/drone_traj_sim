🚁 Drone Trajectory Simulation

📌 Overview

This project demonstrates basic drone control using the pysimverse
simulation environment. The drone performs takeoff, speed adjustment,
and iterative forward and lateral movements to create a diagonal
trajectory pattern.

It serves as a foundational step toward building autonomous drone
systems involving path planning, environment mapping, and computer
vision.

🎯 Features

-   Drone connection and initialization
-   Controlled takeoff to a specified altitude
-   Speed tuning for optimized motion
-   Iterative forward and right movement
-   Approximate diagonal trajectory generation
-   Safe landing execution

🧠 Learning Outcomes

-   Understanding drone control APIs
-   Basics of motion planning in 2D space
-   Sequential vs combined movement concepts
-   Simulation-based testing for robotics
